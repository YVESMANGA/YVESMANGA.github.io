ALLO MBAALIT
Version 0.1



  
      

All Mbaalit est une application mobile créée par un groupe de jeunes développeurs dans le but d'innover le système de gestion de déchets ménagers dans la région de Dakar. En effet, l'objectif visé est d'instaurer dans les quarters dakarois un environnement sain, paisible et durable en y déployant une application; un moyen technologique pour les riverains avec des fonctionnalités telles que savoir où se site le camion et bien d' autres fonctions très utiles. Avec la fonction alerte, vous serez notifié en temps s' il y a un quelconque souci avec le service.
